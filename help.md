## Moderation

**ALL MODERATION IN V4.0 DOES NOT SEND A LOG TO THE USER, BUT INSTEAD TO #case_logs**

ban - bans a user

kick - kicks a user

mute - mutes a user

warn - warns a user

clear (x) - clears a channel of (x) messages - EG: !clear 50
